
## Installation

```bash
$ yarn install
$ yarn add prisma --dev
$ npx prisma migrate dev
Rename example.env >> .env and fill it

```

## Running the app

```bash
# development
$ yarn start:dev

```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
